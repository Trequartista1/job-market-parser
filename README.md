# Job Market Parser (Ukraine)

Web scraper for collecting Data Analyst vacancies from Work.ua.

## Features

- multi-page scraping
- salary parsing
- experience extraction
- city extraction
- dataset generation

## Output dataset

Columns:

- job_id
- title
- company
- city
- salary
- salary_min
- salary_max
- date_posted
- experience_required
- description_preview
- is_hot_job
- link
